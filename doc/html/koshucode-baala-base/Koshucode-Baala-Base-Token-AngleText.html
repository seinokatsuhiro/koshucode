Examples

Simple word.

>>> putStrLn $ angleQuote "aaa"
"aaa"

Text contains line feed.

>>> putStrLn $ angleQuote "aaa\nbbb"
"aaa" #lf "bbb"

Lookup keyword table.

>>> lookup "lf" angleTexts
Just "\n"

Keyword table

Some text literals are written using the following angle text.

<cr>
Carriage return
<crlf>
Carriage return and line feed
<lf>
Line feed
<q>
Single quote (')
<qq>
Double quote (")
<sp>
Space
