Token type

Word
Character sequence not including special characters, e.g., aa, r2d2, 12.0, etc. There are four types of quotations. (1) non-quoted word like aa, (2) single-quoted word like 'aa', (3) double-quoted word like "aa".
Term name
Word beginning with slash, e.g., /aa. Term name like /a/b is used for nested relation, that means term /b in the relation of term /a.
Paren
Open and closed parens. (1) Round parens ( and ), (2) Squared brackets [ and ].
Space
Space characters.
Comment
Texts from double asterisks (**) to end of line are comments. Quadruple asterisks (****) comments are treated in CComment.
