Token type

Word
Character sequence not including special characters, e.g., aa, r2d2, 12.0, etc.
Term name
Word beginning with slash, e.g., /aa. Term name like /a/b is used for nested relation, that means term /b in the relation of term /a.
Paren
Open and closed parens.
Comment
Text from ** to end of line is single line comment.
Space
Space characters.
Line
Source line.

tokenTypeText :: Token -> StringSource

Text of token type, one of "Word", "TermN", "TermP", "Open", "Close", "Space", "Comment", or "Line". 

>>> tokenTypeText $ Word 0 "flower"
"Word"
