operandGroup :: [TokenTree] -> [Named [TokenTree]]Source

Split operand into named group. Non quoted words beginning with hyphen, e.g., -x, are name of group. 

>>> operandGroup $ tokenTrees $ tokens "a b -x c 'd' -y e"
[("",   [TreeL (Word 0 "a"), TreeL (Word 0 "b")]),
 ("-x", [TreeL (Word 0 "c"), TreeL (Word 1 "d")]),
 ("-y", [TreeL (Word 0 "e")])]
