Simple writers are writers with identity string mapping. For example, doc is equivalent for write id.

Examples

Wrap in open and close brackets. 

>>> docWrap "[" "]" "abc"
[abc]

Put spaces between content and brackets. 

>>> docWraps "[" "]" "abc"
[ abc ]
