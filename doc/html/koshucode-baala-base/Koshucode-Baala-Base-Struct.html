Koshucode.Baala.Base.Struct

Description

Runtime structure for calculations written in Koshucode. Module is bundle of calculation stuff.

Dependencies of significant data types.

  • Module uses Module, Assert, Relmap, Judge and ConsRelmap.
  • Assert uses Relmap.
  • Relmap uses Relmap and Rel.
  • ConsRelmap uses ConsHalfRelmap and RelmapFullCons.
  • ConsHalfRelmap uses HalfRelmap and TokenTree.
  • RelmapFullCons uses Relmap and TokenTree.
  • HalfModule uses HalfRelmap, HalfModule, and Token.
  • HalfRelmap uses HalfRelmap and TokenTree.
