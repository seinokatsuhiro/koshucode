type TokenTree = Tree TokenSource

Tree of tokens.

tokenTrees :: [Token] -> [TokenTree]Source

Parse tokens into parened trees. Blank tokens are excluded. 

>>> tokenTrees $ tokens $ "meet (R | pick /a /b)"
[TreeL (Word 0 "meet"),
 TreeB 1 [
   TreeL (Word 0 "R"),
   TreeL (Word 0 "|"),
   TreeL (Word 0 "pick"),
   TreeL (TermN ["/a"]),
   TreeL (TermN ["/b"])]]

There are three types of parens -- 1, 2, or 3. Paren type is in TreeB type subtrees.

  1. Round parens ()
  2. Squared brackets []
  3. Curely braces {}
