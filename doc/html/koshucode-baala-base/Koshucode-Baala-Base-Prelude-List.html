splitBy :: (a -> Bool) -> [a] -> Either [a] ([a], a, [a])Source

Split list by predicate. If list contains an element that satisfies the predicate, (before-list, the-element, after-list) is returned. Otherwise, original list is returned. 

>>> splitBy (== '|') "b c"
Left "b c"

>>> splitBy (== '|') "a | b | c"
Right ("a ", '|', " b | c")
