Text. Int represents quotation level, i.e., 0 for non-quoted, 1 for single-quoted, 2 for double-quoted, 3 for -with variable.

tokenTypeText :: Token -> StringSource

Text of token type, i.e., one of "Text", "Term", "Open", "Close", "Space", or "Comment". 

>>> tokenTypeText $ textToken "flower"
