Token type

data Token Source

There are nine types of tokens.

Constructors

TWord CodePoint Int String

Word. Int represents quotation level, i.e., 0 for non-quoted, 1 for single-quoted, 2 for double-quoted, 3 for -with variable.

TSlot CodePoint Int String

Slot name. Int represents slot level, i.e., 0 for positional slots, 1 for named slots, 2 for global slots.

TShort CodePoint String String

Abbreviated word.

TTerm CodePoint TermPath

Term name.

TOpen CodePoint String

Opening paren.

TClose CodePoint String

Closing paren.

TSpace CodePoint Int

N space characters.

Comment text.

tokenContent :: Token -> StringSource

Get the content of token. 

>>> let tok = TTerm 20 ["/r", "/x"] in tokenContent tok
"/r/x"

tokenTypeText :: Token -> StringSource

Text of token type, i.e., one of "Word", "Term", "Open", "Close", "Space", "Comment", or "Unknown". 

>>> tokenTypeText $ TWord 25 0 "flower"
"Word"
