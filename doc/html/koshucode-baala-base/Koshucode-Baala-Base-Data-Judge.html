Datatype

data Judge c Source

Judgement on type c.

Judgement (or judge for short) is divided into three parts: logical quality, name of pattern, and argument. Boolean values True or False of logical quality corresponds to affirmed or denied judge. A name of judgement pattern represents certain sentence pattern that gives intepretation of data. Sentence pattern has placeholders filled by Named c in argument.
