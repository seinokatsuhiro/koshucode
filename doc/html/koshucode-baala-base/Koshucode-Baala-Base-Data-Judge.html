Datatype

data Judge c Source

Judgement on type c.

Judgement (or judge for short) is divided into three parts: logical quality, sign of relation, and argument. Bool value of logical quality corresponds to affirmed or denied judge. String value of sign represents certain sentence pattern that gives intepretation of data. Sentence pattern has placeholders filled by (String, c) values of argument. -
