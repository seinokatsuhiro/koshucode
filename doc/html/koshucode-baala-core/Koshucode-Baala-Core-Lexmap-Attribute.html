Data type

type Roa = [NamedTrees]Source

Relmap operation attributes as association list.

type Roal a = ((String, Roa), a)Source

Association of operator use and something. Operator use is represented as pair of operator name and attributes.

type RoaSorter = [TokenTree] -> Ab RoaSource

Sorter for attribute of relmap operator. Sorters docompose attribute trees, and give a name to subattribute.

type RoaSpec = (AbMap Roa, [String], [String])Source

Attribute sorter for relmap operator. It consists of trunk sorter, trunk names, and branch names.

Branch sorter

roa :: [TokenTree] -> RoaSource

Split attribute into named group. Non quoted words beginning with hyphen, e.g., -x, are name of group. 

>>> roa $ B.tt "a b -x /c 'd -y e"
[ ("@trunk", [TreeL (TText 1 0 "a"), TreeL (TText 3 0 "b")])
, ("-x", [TreeL (TTerm 7 ["/c"]), TreeL (TText 9 1 "d")])
, ("-y", [TreeL (TText 14 0 "e")]) ]
