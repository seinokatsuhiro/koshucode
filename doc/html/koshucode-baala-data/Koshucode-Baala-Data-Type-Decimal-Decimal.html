Description

Decimal number.

A decimal number in Koshucode is internally represented using a rational number (decimalRatio) with a length of fractional part (decimalFracl) . For example, the decimal number 11.250 correnponds to the rational number 45/4 with the fraction length 3.

Decimal number
Number represented using 10 digits and the decimal separator.
Decimal separator
Symbol between integer part and fractional part.
Fractional part
The right part of decimal numbers, e.g., 250 for 11.250.
Integer part
The left part of decimal numbers, e.g., 11 for 11.250.
Integral decimal
Decimal number without fractional part.
