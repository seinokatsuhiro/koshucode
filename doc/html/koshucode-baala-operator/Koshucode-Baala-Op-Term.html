ropsTerm :: Ord c => [Rop c]Source

Relmap operators for manipulating term names.

cut
Project relation to unspecified terms.
cut-term
Project relation to terms not in relmap output.
pick
Project relation to specified terms.
pick-term
Project relation to terms in relmap output.
rename
Change term name.
move
Change heading.
prefix /P /N ...
Add prefix /P to terms /N ...
prefix-change /P /Q
Change prefix from /P to /Q.
unprefix /P
Remove prefix /P from term name.
