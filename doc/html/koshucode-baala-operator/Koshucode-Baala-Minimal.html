Naming conventions

Functions for relational mapping are named under some conventions.

xxx
Name of relational operator.
ropConsXxx
Functions of RopCons. These functions construct Relmap from operator usages, but constructions are failed if operators are misused.
relmapXxx
Functions from Relmap to Relmap.
relfyXxx
Functions that make Relfy.
likeXxx
Operand parser.
LikeXxx
Operand pattern.
