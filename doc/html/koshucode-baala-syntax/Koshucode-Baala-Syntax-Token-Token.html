Token

data Token Source

There are eleven types of tokens.

Constructors

TText CodePt TextForm String

1) Text.

TSlot CodePt Int String

2) Slot name. Int represents slot level, i.e., 0 for local positional slots, 1 for local named slots, 2 for global slots — @name

TShort CodePt String String

3) Prefixed shorten text — short.proper

TTermN CodePt Ordering TermName

4) Term name — /name

TTerm CodePt TermType TermPath

5) Term path — /name/name

TLocal CodePt (Local String) Int [Token]

6) Local name — ^/name

TOpen CodePt String

7) Opening bracket — (, {, etc

TClose CodePt String

8) Closing bracket — }, ), etc

TSpace CodePt Int

9) N space characters.

TComment CodePt String

10) Comment — **text

TName CodePt BlankName

11) Blank name. (This is used in building content expression)
