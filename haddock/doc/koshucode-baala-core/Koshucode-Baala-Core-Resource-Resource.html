Description

Internal representation of relational data resource. Data resource bundles data themselves and calculation stuffs. In Koshucode, data is represented as set of judgements, and calculation is as relation-to-relation mapping. This internal representation is usually constructed from computer files or web resources addressed by URI. Result of calculation on some resource becomes also data resource.
