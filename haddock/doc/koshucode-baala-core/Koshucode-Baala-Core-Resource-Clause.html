consClause :: [Token] -> SecNo -> [TokenLine] -> [Clause] Source

Convert token list into clause list. Result clause list does not contain CRelmap and CAssert. Instead of them, TTokmap and CAssert are contained. This function does not depend on ConsLexmap.

>>> consClause . B.tokenize $ "a : source A /x /y"
[ TTokmap ( TokenClause
             [TText 1 0 "a", TSpace 2 1, ..., TTerm 11 ["/y"]]
             [CodeLine 1 "a : source A /x /y" [TText 1 0 "a", ...]] )
         "a" [ TText 5 0 "source"
             , TText 7 0 "A"
             , TTerm 9 ["/x"]
             , TTerm 11 ["/y"]
             ]]
