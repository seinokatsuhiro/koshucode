relmapSpecialize :: forall h c. h c -> RelmapLinkTable' h c -> RelkitTable c -> Maybe Head -> Relmap' h c -> Ab (RelkitTable c, Relkit c) Source

Specializes generic relmap to specialized relmap, i.e., Relmap to Relkit. Specialized relmaps have fixed input/output headings. In constrast, generic relmaps do not. Specialized relmaps are also called relkits. This function returns relkit and extended table of relkits.
