copy

consCopy :: RopCons c Source

copy N R

Copy input relation and referenced by ^N in R.

For example, down /r can be rewrite to copy r ( dee | slice /r ^r ).

for

consFor :: CRel c => RopCons c Source

for /P R

Map nested relation at /P by relmap R.

The relamp for /r ( cut /c ) removes term /c from a nested relation at the term /r.

slice

consSlice :: CRel c => RopCons c Source

slice /N [R]

Slice input relation. Each slices have single tuple of input relation, and added as content of term /N to each input tuples. If R is given, slices are mapped by R. If input relation has a nested relation at term /P, the relation can be referenced by ^/P in R.
