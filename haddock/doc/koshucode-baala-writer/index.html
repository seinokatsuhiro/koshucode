koshucode-baala-writer-0.190.0.0: Writers for the Koshu calculator

This package implements writers for the Koshu calculator. Writers serialize internal data of calculator into some formats. Most basic writer is for Koshucode, and other writers are defined for CSV, HTML table, etc.
