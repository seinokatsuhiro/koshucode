chopDigitsTrancate :: (Integral d, Integral n) => d -> n -> n Source

chopDigitsTrancate d n returns a number which does not have the tailing d digits. If d is zero or negative, it returns just n.

chopDigitsRound :: (Integral d, Integral n) => d -> n -> n Source
