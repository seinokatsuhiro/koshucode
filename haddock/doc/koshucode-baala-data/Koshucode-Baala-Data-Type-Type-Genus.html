Constructor

data Genus c Source

Set of various contents. A genus C - X + Y has (1) C is a set of types as representative contents, (2) X is exclusive contents from the set, and (3) Y is inclusive contents to the set.

Contents in X must be in C, contents in Y must not be in C. Internal function genusFrom trims unnecessary contents according to this conditions.
