Description

Term picker is a data for picking target terms based on input heading terms. For example, when picking /a /b from /a /b /c /d /e, use termPicker like:

>>> let pk = termPicker ["a", "b"] ["a", "b", "c", "d", "e"]
>>> pickTerms pk ["1", "2", "3", "4", "5"]
["1", "2"]

termPicker :: (GetTermNames l, GetTermNames r) => l -> r -> TermPicker c Source

Create term picker from left and right term names.

>>> let ss = termPicker ["a", "b", "c"] ["b", "c", "d", "e"]

>>> ssDisjoint ss
False

Left terms.

>>> ssLShareIndex ss
[1, 2]

>>> ssLShareNames ss
["b", "c"]

>>> ssLSide ss "ABC"
"A"

>>> ssLShare ss "ABC"
"BC"

Right terms.

>>> ssRShareIndex ss
[0, 1]

>>> ssRShareNames ss
["b", "c"]

>>> ssRSideNames ss
["d", "e"]

>>> ssRShare ss "BCDE"
"BC"

>>> ssRSide ss "BCDE"
"DE"

>>> ssRSplit ss "BCDE"
("BC", "DE")

>>> ssRAssoc ss "BCDE"
("BC", "BCDE")

>>> ssRForward ss "BCDE"
"BCDE"

>>> ssRBackward ss "BCDE"
"DEBC"
