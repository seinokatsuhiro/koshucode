array

consArray :: CContent c => RopCons c Source

array /P /P ... -to /N E ...
Convert discrete tuples of /P /P ... to term array /N .... The first /P of /P /P is a key term of which content is equal to E, and the second is a value of the key. When E is omitted, term name content is used, e.g., -to /1 /2 means -to /1 '/1 /2 '/2. The length of /P /P ... must be even. Relmap like array /k /v /ka /va -to /1 /1a /2 /2a ... works for multiple key-value pairs.
RELMAP
  array /key /val  -to /1 1  /2 2  /3 3  /4 4
                          :     :     :     :
                          :.....:.....:.....:.... /key

INPUT                    OUTPUT
  /group  /key  /val       /group  /1    /2    /3    /4
  ======= ===== -----      ======= ----- ----- ----- -----
  'a      1     10         'a      10    20    30    ()
  'a      2     20         'b      40    50    ()    ()
  'a      3     30                 .                  .
  'b      1     40                 :..................:
  'b      2     50                 term array
