Extract duplicate elements.

>>> duplicates "banana"

>>> duplicates "grape"

keepMember :: Ord a => [a] -> [a] -> [a] Source

Deprecated: Use selectShare instead.

Filter by set membership. keepMember A B is same to the intersection of A and B, except for the ordering of B is preserved.

>>> keepMember "cba" "abcdefg"
"abc"

>>> keepMember "abcdefg" "cba"
"cba"

omitMember :: Ord a => [a] -> [a] -> [a] Source

Deprecated: Use selectRight instead.

Anti-filter by set membership. omitMember A B is same to B minus fA, except for the ordering of B is preserved.

>>> omitMember "cba" "abcdefg"
"defg"

>>> omitMember "abcdefg" "cba"
""

setList :: Ord a => [a] -> [a] Source

Convert to set-like list, in other words, remove duplicate elements and sort list.

>>> setList "abracadabra"

setEq :: Ord a => [a] -> [a] -> Bool Source

Set-like equality, in other words, duplication and ordering are ignored.

>>> setEq "abc" "bca"
True

>>> setEq "abc" "ab"
False

disjoint :: Eq a => [a] -> [a] -> Bool Source
