picker :: Ord n => [n] -> [n] -> Picker n c Source

Create picker.

data Picker n c Source

Picker data.

For example, /a /b /c and /b /c /d /e have left-proper terms /a, shared terms /b /c, and right-proper terms /d /e.

>>> let pk = picker (words "/a /b /c") (words "/b /c /d /e")
