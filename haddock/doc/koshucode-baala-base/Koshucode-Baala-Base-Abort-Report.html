Description

Report abort reasons.

Abortable process returns Left AbortReason when aborted. The abortable function adds code positions to the reason. abortPrint outputs aborted message in the following format.

** ABORTED  {main reason}
** -------- --------------------------------------------
** Detail   {detailed reason}
**          {detailed reason}
** Source   {line #} {char #} {source}
**          > {code string}
**          {line #} {char #} {source}
**          > {code string}
** Command  {program name}
**          {arg 1}
**          {arg 2}

Print abort message.

Prepare code position and abort reason.

>>> let cp = B.CodePos (B.nioFrom "abcdefg") 1 "abcdefg" "defg"
>>> let Left a = B.abortable "tag" cp $ Left $ B.abortBecause "Bad luck"

Print it.

>>> abortPrint (words "prog x y") a
**
**  ABORTED  Bad luck
**  -------- ---------- ------
**  Source   1 4 <text>
**           > defg     .. tag
**  Command  prog
**           x
**           y
**
